== Описание ==

Плагин «**Витрина товаров Glopart**» – партнерское решение для сайтов на WordPress:

* автоматически сделает вас партнером топовых партнерских программ,
* наполнит ваш сайт WP партнесркими предложениями (landing pages),
* самостоятельно обновит все товары без необходимости совершать дополнительные действия.

== Installation ==

1. Скопируйте папку плагина <strong>product-widget-glopart</strong> в <strong>/wp-content/plugins/</strong>. 
2. Активируйте плагин через меню <strong>Плагины</strong>.
3. Перейдите во вкладку "Внешний вид" - "Виджеты", в поле "Доступные виджеты" найдите виджет "Витрина товаров Glopart" и перенесите его в поле "Область для виджетов".
4. Настройте виджет: придумайте название для вашего блока с товарами Glopart (витрина товаров), укажите ваш Glopart ID ([инструкция по получению Glopart ID](https://youtu.be/h-0MSbYhz_w)), укажите количество отображаемых объявлений в данном блоке, назначьте максимально допустимую ширину картинки в пикселях, выберите выравнивание картинки (слева | над текстом | справа).
5. Это все.
